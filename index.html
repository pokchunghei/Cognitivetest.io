Cognitive Function Test "MBTI"
如果你只想知道所謂“MBTI”的四個字母，
INFJ、ESFP，向別人炫耀「我是I人/我是J人」，
那麽你可以選擇去 16 Personalities 網站，
簡單易懂。

但你真的認為那四個字母就能代表你？

真正的 MBTI (Myers-Briggs Type Indicator)
使用了榮格 (Carl Jung) 所提出的八維為基礎，
包括兩類感知功能 Perceiving function N/S，
和兩類判斷功能 Judging function T/F，
每類功能分別有外傾 E 和內傾 I，
組成八維（Ne, Ni, Se, Si; Te, Ti, Fe, Fi）。

MBTI 並不完全是性格測試，也沒有什麽神奇的成分，
只是透過分析使用者的認知功能，扣連至日常行為，
幫助我們了解自己行為喜好背後的原因，
也有助理解和自己性格相反的人之行為。
